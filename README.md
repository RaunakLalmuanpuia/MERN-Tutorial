# Notes App (MySQL + Express + React + Node.js)

A simple full-stack Notes app using React, Express, Node.js, and MySQL. Users can create notes, like notes, add comments, manage their profile, and change their password.

## Features
- Create & view notes
- Like a note
- Add comments
- User profile view/update
- Change password
- MySQL for storage, Express API, React frontend

## Setup
Install dependencies in both `backend` and `frontend`, configure `.env` for MySQL, then run:
```bash
npm start    # backend
npm start  # frontend
